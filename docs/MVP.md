# Never Miss Twice — MVP タスクリスト

## 習慣の管理

- [ ] 習慣を追加できる
- [ ] 習慣を削除できる
- [ ] 登録した習慣の一覧を見られる

## 毎日の記録

- [ ] 今日の習慣を完了にマークできる
- [ ] 完了を取り消せる

## ストリークの確認

- [ ] 各習慣の連続達成日数を確認できる
- [ ] 昨日ミスした習慣に警告が表示される（今日やれば継続できる）
- [ ] 2日連続でミスしたらストリークがリセットされる

## 通知・バッジ

- [ ] 今日やらないと継続が途切れる習慣がある場合、Dockアイコンに件数が表示される

## データの保持

- [ ] アプリを再起動してもデータが消えない
